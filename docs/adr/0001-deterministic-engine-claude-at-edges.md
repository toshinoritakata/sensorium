# 計算は決定論ルールエンジン、Claude は入口と出口のみ

成立条件と機材案の算出は、機材DBを真実とする決定論的ルールエンジンが担う。Claude は入口（自然言語の構想 → InteractionSpec の抽出）と出口（成立条件・機材案 → レポート文章の生成）のみに使う。

What-if（Spec / Setup を変えるたびの再計算）が中核機能であるため、計算をエンジンに置くことで即時・再現可能・オフライン動作・課金なしの再計算を得る。判定に Claude を使う柔軟さは捨てるが、その代償として速度・再現性・オフライン性を取った。後からこの境界を動かすと What-if の性質が根本から崩れるため、明文化して固定する。
